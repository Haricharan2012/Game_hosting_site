Xgames-Xtreme games unlimited

our aim is to provide unique and distinct games to Gamers of all ages so that there is something interesting and special for everyone,it would also bridge the gap between avid gamers and casual gamers and bring out the absolute significance of a game which is to have 'fun'

The aim is to make a unique set of flash games. In which the player/user would be able to choose from a variety of distinct games and those said games would have the ability to run on any kind of modern devices like laptops,smartphones,tablets,TV's etc.. this is possible due the fact that every modern device today has a browser application and these set of games run on a browser. this in turn reduces the hardware requirements needed to play the games as well as brings great versatility as the player/user can play their favorite game anytime,anywhere without any trouble,along with the games being cross platform it would also have a unique feature that is to run in an offline mode. As a great assortment of games exists there is something for everyone to play and enjoy. All The games would contain database connectivity through which after the completion of the games the player would be prompted by a text box to enter his/her name which would be stored along with the earned scores in the database. the database would show a list of high scores of different players in a specific order.
